# ROS2 packaging with cmake and RPM

ROS2 cmake packaging example

# build

```bash
make build
```

# package

```bash
make package
```

# run

```bash

. install/setup.bash

ros2 run publisher publisher_node
ros2 run subscriber subscriber_node

```
